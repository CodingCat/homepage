Bring BigData to Everyone

Bring BigData to Everyone

We are developing a user-friendly Machine Learning system, so that

  • You can analyze large volume of data efficiently, even you never learn programming stuffs

  • You can evaluate your new algorithms, even you are a machine learning expert but with limited system background

Multiple-level abstraction to describe machine learning task workflow

Existing large-scale computing frameworks only provide unique abstraction to describe computation tasks. Some of them, like Hadoop [1] and Spark [2] , require users to give detailed implementation of Map() and Reduce() functions, implicitly requiring users to be experienced Java/Scala programmers; Hive/Shark enable users to query data with SQL-like languages, but makes machine learning researchers cannot build customized learning algorithms.

In this part, our goal is to provide multiple level abstractions to describe the workflow of machine learning algorithms. For users without technical background, they can use high-level description language to perform classification or regression on the dataset, for example, when performing SVM classification:

val X = load("input_set")

val Y = load("label_set")

val model = classify("SVM", X, Y)

For machine learning researchers, we provide operator-level abstraction for more flexibility to build ad-hoc algorithms, e.g. gradient descending or matrix operation. Machine learning researchers can connect the operators through the GUI or model description languages.
