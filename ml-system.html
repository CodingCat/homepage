Bring BigData to Everyone

Bring BigData to Everyone

We are developing a user-friendly Machine Learning system, so that

  • You can analyze large volume of data efficiently, even you never learn programming stuffs

  • You can evaluate your new algorithms, even you are a machine learning expert but with limited system background

Introduction

Machine learning (ML) is one of the key techniques to transform raw datasets into actionable knowledge. With the growing size of data, the barrier for users to handle such large volume of data is becoming higher. Our goal is to develop a powerful and user-friendly machine learning system.

ML Computation Model

Existing distributed computation frameworks focus on providing low level primitives for users to implement concurrent algorithms. Our goal is to implement those common components in ML algorithms in a distributed manner and provide them to users as primitives for modeling ML algorithms efficiently.

Unified Runtime Environment

Users have requirement on analyzing historical as well as real-time data. We provide unified runtime environment to run these two kinds of jobs efficiently.

Unified Data Accessing Interface

In many cases, data to be analyzed are stored in different systems which provides multiple sets of APIs. It's not practical to expect a common user to master all of them. We are providing a unified interface which supports accessing data from different data sources.

Multiple-level Abstractions

Users with different background require abstractions on different levels to describe machine learning tasks. We are providing abstractions from operator (e.g. matrix operations) to learning model (e.g. linear regression) level to support running existing algorithms as well as developing new algorithms.

Automatic Optimization Engine

The optimization engine is to reduce IO overhead in learning process by caching hot data in memory and avoid the exploration in large search space transparently.
